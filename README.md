# Projeto Stem Criar ⚙️

## Sobre o projeto

O **projeto Stem Criar** é um Projeto de P&D desenvolvido pela EST/UEA, incentivado pela Lei da Informática para a Amazônia Ocidental (Lei Federal n° 8.387/1991). 

O Stem Criar promove **capacitação gratuita** de professores e alunos da rede pública de ensino no interior e na capital do estado do Amazonas nas áreas de:

* Fabriação Digital;
* Robótica;
* Programação;
* Eletrônica Básica;

A partir de uma **metodologia maker** de ensino, as atividades do projeto estimulam **a aplicação do conhecimento em situações do mundo real**. Assim, buscamos transformar a educação para ser mais inovadora, atrativa e criativa!
